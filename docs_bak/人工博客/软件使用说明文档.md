### 软件使用说明文档

### 1、使用步骤

+ 解压到系统盘符的任意目执行录
+ 右键执行文件夹中的 start.bat

### 2、软件目录说明

![](https://cdn.jsdelivr.net/gh/chen-xing/figure_bed/cdn/20210205150105566.png)

| 目录          | 说明                                   |
| ------------- | -------------------------------------- |
| config        | 系统的配置目录                         |
| jre           | 软件是java开发的需要java的运行环境环境 |
| logs          | 软件运行过程中产生的日志               |
| datamover.jar | 软件核心的程序                         |
| start.bat     | 软件的启动脚本                         |
